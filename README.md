# Grade Verification System

A client-side React + TypeScript web app for verifying spreadsheet-based grade computations.

## Stack

- React
- TypeScript
- Tailwind CSS
- SheetJS (`xlsx`)

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Required Spreadsheet Columns

- `No.`
- `Last Name`
- `First Name`
- `Prelim`
- `Midterm`
- `Final Term`
- `Final Grade`
